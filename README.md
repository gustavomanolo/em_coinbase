# Coinbase websocket Feed

### Run dev
```sh
$ git clone https://github.com/gustavomanolo/em_coinbase.git
$ cd em_coinbase
$ npm run dev
$ open web browser on http://localhost:3000/
```

### Run production
```sh
$ git clone https://github.com/gustavomanolo/em_coinbase.git
$ cd em_coinbase
$ npm run build
$ npm run start
$ open web browser on http://localhost:3000/
```
